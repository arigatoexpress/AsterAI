# Aster API Credentials Template
# ================================

# Your API credentials should look similar to these examples:

API Key Example:
- Format: Usually starts with 'ak_' or similar prefix
- Length: Typically 32-64 characters
- Example: ak_1a2b3c4d5e6f7g8h9i0j1k2l3m4n5o6p

API Secret Example:
- Format: Usually starts with 'sk_' or similar prefix
- Length: Typically 32-64 characters
- Example: sk_9z8y7x6w5v4u3t2s1r0q9p8o7n6m5l4k

# Where to find them:
1. Log in to https://aster.exchange (or your Aster platform URL)
2. Navigate to Settings > API Keys
3. Create a new API key if needed
4. Copy both the API Key and Secret

# Security Notes:
- Never share your API Secret
- Store credentials securely
- Use read-only keys for testing
- Enable IP whitelisting if available

# When ready, run:
python scripts/update_api_keys.py

# And paste your actual credentials when prompted
